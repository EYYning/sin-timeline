```

1938-2025     1938   1964       2025    2025    |   클릭→인스타
               ────────── 14개 전부 ──────────
```

## 🚀 빠른 시작

### 1. 프로젝트 생성

```bash
npm create vite@latest sin-timeline -- --template react
cd sin-timeline
npm install
npm install lucide-react
```

### 2. 파일 추가

- `SinNewspaperTimeline.jsx` → `src/` 폴더
- `sin1.png ~ sin14.png` (14개) → `public/` 폴더
- `final-poster.png` (1개) → `public/` 폴더
- `gridshift.mp3` → `public/` 폴더

### 3. App.jsx 수정

```jsx
import SinNewspaperTimeline from './SinNewspaperTimeline'

function App() {
  return <SinNewspaperTimeline />
}

export default App
```

### 4. 실행

```bash
npm run dev
```

## 🎨 주요 기능

### 1. 수평 스크롤 타임라인
- sin1-14 전부 좌→우 일렬 배치 (14개)
- 마우스 휠로 좌우 스크롤

### 2. 최종 포스터 (별도)
- 타임라인 끝에 디바이더로 구분
- 클릭 → 인스타그램 링크 팝업
- 500×700px 크기

### 3. 인터랙티브 호버
- 각 신 글자 + 최종 포스터 호버 시 상세 정보
- 좌측 하단 정보 패널

### 4. 그리드쉬프트 오디오
- 우측 하단 플레이어
- 1300+ BPM 리듬

### 5. 인터뷰 탭 시스템
- 5초 후 첫 등장
- 30초마다 새 인터뷰
- 60초 후 자동 삭제
